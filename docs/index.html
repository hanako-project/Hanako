Project Hanako

Formerly, this was ProjectAmbis, an abstract project that i want to make but it was too abstract that i haven't any idea where to start, now its here

What is This ?

For now, this is a RESTful Webservice API for an anime database

Usage

Access this from on wherever your deployment is, in this example, on this domain

hanako.luqmansen.me/api/v1/user/new
hanako.luqmansen.me/api/v1/user/login
hanako.luqmansen.me/api/v1/anime/all
hanako.luqmansen.me/api/v1/anime/{id}
hanako.luqmansen.me/api/v1/anime/search

Query keyword

/api/v1/anime/search/q?title=anime_to_find
/api/v1/anime/all?show=1000
